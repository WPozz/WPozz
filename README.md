## 📂 Featured Projects
* **Multi-class-classification-using-Trasformers-for-medical-text([Link to repo](https://github.com/WPozz/Multi-class-classification-using-Trasformers-for-medical-text-)):** The project classifies clinical transcriptions using RNN and Bio_ClinicalBERT models, leveraging Back Translation and Summary Augmentation to address class imbalance. The best-performing model, Bio_ClinicalBERT with Focal Loss (γ=4) trained on the Summary Augmentation dataset, achieved the optimal balance between sensitivity to rare classes and overall precision. Built with python and PyTorch. 
* **Diabetes Prediction with Support Vector Machines([Link to repo](https://github.com/WPozz/Support-Vector-Machine-Classifier-for-Diabetes)):** This repository contains a comprehensive machine learning project for predicting diabetes using various Support Vector Machine (SVM) models. The project follows a complete end-to-end workflow, from data analysis and model training to in-depth interpretation, clinical evaluation, and deployment preparation. Built with Python and Sklearn.
